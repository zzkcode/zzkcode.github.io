康奈尔笔记法

2019-07-06

最大作用

  • 站在宏观角度总结知识点
  • 鼓励你养成以后咀嚼的习惯
  • 对知识点进行回顾

优点

便于制作：即使是普通笔记本，画三条线也就可以开始了

步骤

  1. 将笔记本分成三部分（左1，右2，下3）
    1. 记录时只写在2号区域
    2. 使用符号，善用缩写，言简意赅，列要点，每点之间空行
    3. 在右上角加上主题和日期
  2. 课后总结右侧笔记要点或问题写在左侧 即写在1号区域，复习时，遮住笔记，只看要点提示，复述出所学内容。用自己的话将要点总结在页面下方。
  3. 多次记忆 每周至少花10分钟（时间根据自己的来），重读笔记，再次按要点复述并记忆
  4. 反思 这些知识和观点的意义是什么？它们的依据是什么？我怎么把它们应用到我已经知道的事物当中？还有什么是我可能没想到，别人没涉及的？
